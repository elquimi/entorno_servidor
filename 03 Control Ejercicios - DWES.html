    03 Control Ejercicios

    Interactuando con el Usuario: Formularios HTML y PHP

    Para que nuestros ejercicios sean interactivos, necesitamos una forma de que el usuario introduzca datos. La manera más común es a través de formularios HTML. Un formulario es un conjunto de campos (texto, botones, etc.) que el usuario puede rellenar. Cuando pulsa el botón de envío, los datos viajan a un script de PHP para ser procesados.

    ¿Cómo funciona?

    1. El Fichero HTML (.html): Contiene la estructura del formulario. La etiqueta <form> es la más importante. Sus atributos clave son:

      • action: Especifica la URL del script PHP que recibirá los datos.
      • method: Indica cómo se enviarán los datos. Los dos métodos más comunes son:
        • GET: Los datos se añaden a la URL. Es visible para el usuario (ej: procesar.php?nombre=Juan&edad=20). Es útil para búsquedas o cuando quieres que la URL se pueda guardar en marcadores.
        • POST: Los datos se envían en el cuerpo de la petición HTTP, de forma oculta. Es más seguro para enviar información sensible como contraseñas.

    2. El Fichero PHP (.php): Contiene la lógica para procesar los datos. PHP pone los datos recibidos en unas variables globales especiales:

      • $_GET: Un array asociativo con los datos enviados por el método GET.
      • $_POST: Un array asociativo con los datos enviados por el método POST.
      • $_REQUEST: Un array que contiene los datos de $_GET, $_POST y $_COOKIE. Es cómodo pero menos específico.

    Ejemplo Práctico

    Vamos a crear un formulario que pide un número y un script PHP que calcula su cuadrado.

    ARCHIVO formulario.html 

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Calculadora de Cuadrados</title>
</head>
<body>
    <h2>Introduce un número para calcular su cuadrado</h2>
    <form action="calcular.php" method="POST">
        <label for="numero">Número:</label>
        <input type="number" id="numero" name="numero_usuario">
        <br><br>
        <input type="submit" value="Calcular">
    </form>
</body>
</html>

    ARCHIVO calcular.php 

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Resultado</title>
</head>
<body>
    <h1>Resultado del Cálculo</h1>
    <?php
        // Comprobamos si la variable 'numero_usuario' ha sido enviada
        if (isset($_POST["numero_usuario"])) {
            // Recuperamos el número del formulario.
            // Lo convertimos a entero para asegurar que es un número.
            $n = (int)$_POST["numero_usuario"];

            // Calculamos el cuadrado
            $cuadrado = $n * $n;

            // Mostramos el resultado
            echo "<p>El cuadrado de $n es: <strong>$cuadrado</strong></p>";
        } else {
            echo "<p>No se ha recibido ningún número. Por favor, vuelve al formulario.</p>";
        }
    ?>
    <a href="formulario.html">Volver a calcular</a>
</body>
</html>

    En los siguientes ejercicios, aplicarás esta técnica para recibir datos y procesarlos usando bucles.

    Ejercicios de Consolidación

    1) El Generador de Tablas de Multiplicar | Nivel: Fácil

    Objetivo

    Crear un script que genere la tabla de multiplicar de un número proporcionado por el usuario, practicando el uso del bucle for.

    Tarea a realizar
    1. Crea un fichero index.html con un formulario que pida al usuario un número. El formulario enviará los datos a tabla.php mediante el método GET.
    2. En tabla.php, recupera el número enviado.
    3. Utiliza un bucle for para iterar del 1 al 10.
    4. En cada iteración, calcula y muestra una línea de la tabla de multiplicar, por ejemplo: "5 x 3 = 15".
    5. Muestra los resultados dentro de una tabla HTML para una mejor presentación.
    Aplicación en el Mundo Real

    Generar informes dinámicos, listados o calendarios donde se repite un patrón numérico es una tarea común. Por ejemplo, calcular plazos de pago mensuales o proyecciones financieras.

    2) El Sumatorio Automático | Nivel: Fácil

    Objetivo

    Calcular la suma de todos los números enteros desde 1 hasta un número N, utilizando un bucle while.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número entero positivo (N) y lo envíe a un script PHP mediante POST.
    2. En el script PHP, inicializa una variable para la suma total en 0 y una variable contador en 1.
    3. Usa un bucle while que se ejecute mientras el contador sea menor o igual que N.
    4. En cada iteración, añade el valor del contador a la suma total e incrementa el contador.
    5. Al finalizar el bucle, muestra el resultado de la suma total.
    Aplicación en el Mundo Real

    Este tipo de cálculo acumulativo es fundamental en estadística (calcular medias), finanzas (cálculo de balances acumulados) y en la optimización de algoritmos.

    3) ¡Contraseña, por favor! | Nivel: Fácil

    Objetivo

    Simular un sistema de validación simple usando un bucle do-while, donde se solicita una acción hasta que se cumple una condición.

    Tarea a realizar

    Vamos a simular un proceso que necesita un "token" para continuar. 1. Crea un formulario que pida un token de acceso (será un número). Envíalo a un script PHP. 2. El token correcto será 1234. 3. En el script PHP, recibe el token. Utiliza una estructura do-while (simulada, ya que no podemos hacer un bucle real que espere input del usuario en cada iteración). 4. La idea es que si el token es incorrecto, muestres un mensaje de error y el formulario de nuevo. Si es correcto, muestras un mensaje de "Acceso Concedido". 5. Para este ejercicio, lo implementaremos todo en un solo fichero PHP que se envía a sí mismo.

    Aplicación en el Mundo Real

    Los bucles do-while son útiles en validación de entradas, menús interactivos en consola o procesos que deben ejecutarse al menos una vez antes de comprobar la condición de salida.

    4) El Contador de Pares | Nivel: Fácil

    Objetivo

    Practicar el bucle for con un incremento diferente de 1.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número entero N superior a 10.
    2. En el script PHP, recibe N.
    3. Usa un bucle for para mostrar todos los números pares desde 2 hasta N.
    4. Para ello, en lugar de incrementar el contador de 1 en 1 ($i++), increméntalo de 2 en 2 ($i += 2).
    Aplicación en el Mundo Real

    Recorrer colecciones de datos saltando elementos (por ejemplo, procesar solo las filas pares de un fichero), o en algoritmos que operan sobre índices específicos.

    5) Arte ASCII: El Cuadrado Perfecto | Nivel: Fácil

    Objetivo

    Entender el funcionamiento de los bucles anidados para generar patrones bidimensionales.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número N (entre 3 y 15) que representará el lado de un cuadrado.
    2. En el script PHP, usa un bucle for externo que controle las filas.
    3. Dentro de él, anida otro bucle for que controle las columnas.
    4. El bucle interno imprimirá un asterisco (*) N veces en la misma línea.
    5. El bucle externo, después de que el interno termine, imprimirá un salto de línea (<br>).
    6. Utiliza una fuente monoespaciada (como monospace o Courier) para que los caracteres se alineen correctamente.
    Aplicación en el Mundo Real

    Los bucles anidados son esenciales para procesar estructuras de datos multidimensionales como matrices, tablas, o imágenes (recorrer píxeles por filas y columnas).

    6) La Fábrica de Factoriales | Nivel: Fácil

    Objetivo

    Calcular el factorial de un número usando un bucle for y un acumulador.

    Tarea a realizar
    1. Crea un formulario para que el usuario introduzca un número entero no negativo.
    2. El factorial de un número n (escrito n!) es el producto de todos los enteros positivos menores o iguales a n. Ejemplo: 5! = 5 * 4 * 3 * 2 * 1 = 120. El factorial de 0 es 1.
    3. En tu script PHP, inicializa una variable factorial en 1.
    4. Usa un bucle for que vaya desde n hasta 1.
    5. En cada iteración, multiplica la variable factorial por el valor actual del contador.
    6. Muestra el resultado final.
    Aplicación en el Mundo Real

    El cálculo de factoriales es fundamental en combinatoria y probabilidad, áreas de las matemáticas con aplicación en análisis de datos, criptografía y algoritmos.

    7) ¡Encuentra el Bug! La Cuenta Atrás Infinita | Nivel: Fácil

    Objetivo

    Desarrollar habilidades de depuración identificando y corrigiendo un error común en la condición o el iterador de un bucle.

    Setup Inicial

    El siguiente código intenta mostrar una cuenta atrás desde 10 hasta 1, pero contiene un error que provoca un bucle infinito. Tu navegador probablemente se bloqueará o dará un error de tiempo de ejecución.

    <!DOCTYPE html>
<html lang="es">
<head><title>Cuenta Atrás Rota</title></head>
<body>
    <h1>¡Despegue en...</h1>
    <?php
        // Bucle con error
        for ($i = 10; $i > 0; $i++) {
            echo "$i... <br>";
        }
        echo "¡Despegue!";
    ?>
</body>
</html>
    Tarea a realizar
    1. Copia el código en un fichero PHP.
    2. Analiza la estructura del bucle for: la inicialización ($i = 10), la condición ($i > 0) y el iterador ($i++).
    3. Identifica por qué la condición para detener el bucle nunca se cumple.
    4. Corrige el error para que el programa imprima correctamente la cuenta atrás de 10 a 1.
    Aplicación en el Mundo Real

    La depuración es una de las tareas más importantes de un programador. Un error de lógica en un bucle puede causar desde un bloqueo del programa hasta consumir todos los recursos del servidor.

    8) Adivina el Número Secreto | Nivel: Fácil

    Objetivo

    Integrar la generación de números aleatorios con la lógica de bucles para crear un juego simple.

    Setup Inicial

    PHP tiene una función muy útil para generar números aleatorios: rand(min, max). * Propósito: Devuelve un número entero aleatorio entre min y max (ambos incluidos). * Ejemplo: echo rand(1, 10); podría imprimir 7, 1, 10 o cualquier otro número en ese rango.

    Tarea a realizar

    Vamos a crear un juego de "Adivina el número" en un solo fichero PHP. 1. El script generará un número aleatorio entre 1 y 100 y lo "recordará". Para ello, cuando se muestre el formulario por primera vez, generaremos el número y lo guardaremos en un campo oculto (<input type="hidden">). 2. El formulario pedirá al usuario que introduzca su intento. 3. Al enviar el formulario, el script PHP comparará el intento del usuario con el número secreto (que recuperará del campo oculto). 4. Mostrará una de estas tres pistas: "Demasiado alto", "Demasiado bajo" o "¡Correcto! ¡Has adivinado!". 5. Si el usuario no ha acertado, se volverá a mostrar el formulario para que pueda intentarlo de nuevo, pero manteniendo el mismo número secreto.

    Aplicación en el Mundo Real

    Esta lógica de comparación y repetición se usa en sistemas de autenticación (verificar PIN), concursos, algoritmos de búsqueda y cualquier proceso que requiera prueba y error.

    Ejercicios de Refuerzo

    9) Arte ASCII: La Pirámide | Nivel: Medio

    Objetivo

    Profundizar en el uso de bucles anidados donde la condición del bucle interior depende del iterador del bucle exterior.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número para la altura de un triángulo (por ejemplo, 5).
    2. En el script PHP, utiliza un bucle for externo para las filas, que irá de 1 hasta la altura introducida.
    3. Dentro, anida otro bucle for para las columnas. La clave es que este bucle interno debe ejecutarse tantas veces como el número de la fila actual.
    4. Por ejemplo, en la fila 1, imprime 1 asterisco. En la fila 2, imprime 2 asteriscos, y así sucesivamente.
    5. Recuerda usar una fuente monoespaciada para la correcta visualización.
    6. Pista: La condición del bucle interno podría ser $col <= $fila.
    Aplicación en el Mundo Real

    Generación de informes con agrupaciones variables, renderizado de gráficos (como histogramas) o algoritmos de ordenación como el "Bubble Sort" que realizan pasadas cada vez más cortas sobre un conjunto de datos.

    10) El Desafío de Euler: Múltiplos de 3 o 5 | Nivel: Medio

    Objetivo

    Combinar bucles con sentencias condicionales (if) y el operador módulo (%) para resolver un problema lógico clásico.

    Tarea a realizar

    Este es el primer problema del famoso Proyecto Euler. 1. Crea un formulario que pida un número N. 2. En tu script PHP, calcula la suma de todos los números naturales por debajo de N que son múltiplos de 3 o de 5. 3. Por ejemplo, si N es 10, los números son 3, 5, 6 y 9. La suma es 23. 4. Utiliza un bucle for para iterar desde 1 hasta N-1. 5. Dentro del bucle, usa una condición if para comprobar si el número actual es múltiplo de 3 O (||) múltiplo de 5. 6. Pista: Para saber si un número $i es múltiplo de 3, la condición es $i % 3 == 0.

    Aplicación en el Mundo Real

    Filtrado de datos. Por ejemplo, en una tienda online, seleccionar todos los productos que pertenecen a la categoría "electrónica" O tienen un descuento superior al 20%.

    11) Potencia Manual | Nivel: Medio

    Objetivo

    Implementar una operación matemática (potenciación) usando un bucle, sin utilizar el operador ** o la función pow().

    Tarea a realizar
    1. Crea un formulario que pida dos números: la base y el exponente (entero no negativo).
    2. En el script PHP, recibe ambos números.
    3. Calcula base elevado al exponente usando un bucle.
    4. Inicializa una variable resultado en 1.
    5. Utiliza un bucle for que se repita tantas veces como indique el exponente.
    6. En cada iteración, multiplica el resultado por la base.
    7. Maneja el caso especial de que el exponente sea 0 (el resultado siempre es 1).
    8. Pista: El bucle puede ir desde 1 hasta el valor del exponente.
    Aplicación en el Mundo Real

    Entender cómo se implementan las operaciones matemáticas a bajo nivel es clave en programación de sistemas, desarrollo de videojuegos (cálculos de físicas) o criptografía, donde a menudo se deben reimplementar funciones por eficiencia o seguridad.

    12) Las Tablas de Multiplicar al Completo | Nivel: Medio

    Objetivo

    Generar una estructura de datos bidimensional (una tabla HTML) usando bucles anidados.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número N (por ejemplo, 10).
    2. Tu script PHP deberá generar una tabla de multiplicar completa, desde la tabla del 1 hasta la tabla del N.
    3. La salida debe ser una tabla HTML.
    4. Usa un bucle for externo para las filas, que representará cada número del 1 al N.
    5. Usa un bucle for interno para las columnas, que representará el multiplicador del 1 al 10.
    6. La celda (fila, columna) contendrá el resultado de fila * columna.
    Aplicación en el Mundo Real

    Esta estructura es la base para generar cualquier tipo de cuadrícula o matriz en una interfaz de usuario: calendarios, tableros de ajedrez, hojas de cálculo, horarios, etc.

    13) El Guardián de los Primos | Nivel: Medio

    Objetivo

    Implementar un algoritmo para determinar si un número es primo, combinando bucles y lógica condicional.

    Tarea a realizar
    1. Crea un formulario que pida al usuario un número entero positivo.
    2. Un número es primo si solo es divisible por 1 y por sí mismo. (Ej: 2, 3, 5, 7, 11).
    3. En tu script PHP, implementa la lógica para comprobar si el número es primo.
    4. Asume que el número es primo ($es_primo = true).
    5. Usa un bucle for que vaya desde 2 hasta la raíz cuadrada del número. ¿Por qué la raíz cuadrada? Porque si un número n tiene un divisor mayor que su raíz cuadrada, obligatoriamente debe tener otro divisor menor.
    6. Dentro del bucle, comprueba si el número es divisible por el contador actual. Si lo es, cambia la variable $es_primo a false y rompe el bucle con break, ya que no hace falta seguir comprobando.
    7. Al final, informa al usuario si el número era primo o no.
    8. Pista: La función para la raíz cuadrada es sqrt(). Ejemplo: $raiz = sqrt(25);
    Aplicación en el Mundo Real

    Los números primos son la base de la criptografía moderna (como el algoritmo RSA) que asegura las comunicaciones en internet. Los algoritmos de primalidad son cruciales en ciberseguridad.

    14) El Validador Exigente | Nivel: Medio

    Objetivo

    Crear un formulario que se valida a sí mismo, mostrando mensajes de error hasta que la entrada del usuario es correcta, simulando un bucle de validación.

    Tarea a realizar
    1. Crea un único fichero PHP que contenga un formulario y la lógica de validación.
    2. El formulario pedirá al usuario que introduzca su edad.
    3. La lógica de validación es la siguiente: la edad debe ser un número entre 18 y 65.
    4. Si el formulario no ha sido enviado, o si la edad es incorrecta, se muestra el formulario. Si hay un error, se debe mostrar un mensaje descriptivo (ej: "La edad debe estar entre 18 y 65").
    5. Si la edad es correcta, se muestra un mensaje de éxito (ej: "Edad validada correctamente: 30 años").
    6. Pista: Puedes usar la variable superglobal $_SERVER["REQUEST_METHOD"] para saber si el formulario se ha enviado.
    Aplicación en el Mundo Real

    La validación de datos de entrada es una de las tareas más importantes en cualquier aplicación web para garantizar la integridad de los datos y la seguridad. Todos los formularios de registro, login o compra la utilizan masivamente.

    Ejercicios de Ampliación

    15) El Algoritmo de Euclides | Nivel: Alto

    Objetivo

    Implementar un algoritmo matemático clásico y eficiente para encontrar el máximo común divisor (MCD) de dos números, demostrando un uso avanzado del bucle while.

    Tarea a realizar
    1. Crea un formulario que pida al usuario dos números enteros positivos, a y b.
    2. Implementa el algoritmo de Euclides para encontrar el MCD:
      • Mientras b no sea cero, haz lo siguiente:
        • Guarda el valor de b en una variable temporal, temp.
        • Actualiza b con el resto de la división de a entre b (a % b).
        • Actualiza a con el valor de temp.
      • Cuando el bucle termina, a contiene el MCD.
    3. Muestra el resultado al usuario.
    Aplicación en el Mundo Real

    El algoritmo de Euclides es uno de los más antiguos y eficientes. Se utiliza en criptografía (en el algoritmo RSA), en la simplificación de fracciones y en la resolución de ecuaciones diofánticas.

    16) Arte ASCII: El Diamante de Asteriscos | Nivel: Alto

    Objetivo

    Dominar los bucles anidados complejos para crear patrones simétricos, controlando tanto los caracteres impresos como los espacios en blanco.

    Tarea a realizar
    1. Crea un formulario que pida un número impar N que será la anchura máxima del diamante.
    2. El diamante se compone de dos partes: una pirámide superior (creciente) y una pirámide inferior invertida (decreciente).
    3. Parte Superior:
      • Necesitarás un bucle externo para las filas.
      • Dentro, un bucle para imprimir los espacios en blanco iniciales (el número de espacios disminuye a medida que la fila aumenta).
      • Y otro bucle para imprimir los asteriscos (el número de asteriscos aumenta en 2 en cada fila).
    4. Parte Inferior:
      • Similar a la superior, pero invirtiendo la lógica: los espacios aumentan y los asteriscos disminuyen.
    5. Es un reto que requiere planificar cuidadosamente las variables y condiciones de cada bucle.
    Aplicación en el Mundo Real

    Este tipo de control preciso sobre la salida es fundamental en la generación de interfaces de texto (TUI), visualización de datos en consola, y en algoritmos gráficos para renderizar formas y patrones.

    17) Simulador de Inversión a Plazo Fijo | Nivel: Alto

    Objetivo

    Aplicar bucles para modelar un proceso financiero a lo largo del tiempo, mostrando una proyección año a año.

    Tarea a realizar
    1. Crea un formulario que pida tres datos:
      • Capital inicial (ej: 1000).
      • Tasa de interés anual (en %, ej: 5).
      • Número de años de la inversión (ej: 10).
    2. En el script PHP, calcula y muestra una tabla con la evolución de la inversión año por año, usando interés simple.
    3. La tabla debe tener tres columnas: "Año", "Interés Ganado" y "Balance Final".
    4. Usa un bucle for que itere desde el año 1 hasta el número de años especificado.
    5. En cada iteración:
      • Calcula el interés ganado ese año: interes = capital_inicial * (tasa / 100).
      • El capital para el cálculo del siguiente año no cambia (es interés simple).
      • Calcula el balance acumulado: balance_acumulado = capital_inicial + (interes * año_actual).
      • Muestra una fila en la tabla con los datos del año actual.
    Aplicación en el Mundo Real

    Los modelos financieros son una aplicación directa de los bucles. Se usan para calcular amortizaciones de hipotecas, planes de pensiones, proyecciones de crecimiento de negocio y cualquier escenario que evolucione en el tiempo a intervalos regulares.

    18) El Enigma de la Conjetura de Collatz | Nivel: Alto

    Objetivo

    Implementar un algoritmo basado en una famosa conjetura matemática y realizar una revisión de código por pares para evaluar la claridad y corrección de la solución.

    Tarea a realizar

    1. Resolución del Problema:

      • Crea un formulario que pida un número entero positivo n.
      • La Conjetura de Collatz dice que si aplicas repetidamente las siguientes reglas a n, eventualmente llegarás a 1:
        • Si n es par, el siguiente número es n / 2.
        • Si n es impar, el siguiente número es 3 * n + 1.
      • Escribe un script PHP que, dado un n inicial, muestre la secuencia completa hasta llegar a 1.
      • Ejemplo: Si n = 6, la secuencia es: 6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1.
      • Usa un bucle while que continúe mientras n sea mayor que 1.

    2. Revisión por Pares (Peer Review):

      • Una vez que tu solución funcione, intercámbiala con un compañero.
      • Analiza el código de tu compañero y responde a las siguientes preguntas por escrito:
        1. ¿El código resuelve el problema correctamente? Pruébalo con los números 6, 19 y 27.
        2. ¿El código es fácil de leer? ¿Las variables tienen nombres claros? ¿Está bien comentado?
        3. ¿La salida está bien formateada y es fácil de entender para el usuario?
        4. Sugiere al menos una posible mejora (aunque sea pequeña) en la claridad, eficiencia o formato del código.
    Aplicación en el Mundo Real

    La revisión de código es una práctica estándar en el desarrollo de software profesional. Ayuda a detectar errores, mejorar la calidad del código, compartir conocimiento y asegurar que el software sea mantenible a largo plazo.
